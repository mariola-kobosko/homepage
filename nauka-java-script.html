Czym jest JavaScript?

JavaScript to nie to samo co JAVA (!!!)
Pliki zawierające kod Java Script mają rozszerzenie . js oraz mają swój znacznik script, który umieszczamy na końcu strony, przed zamknięciem znacznika body
Java Script odpowiada za interakcje na stronie

Implementuje rzeczy ze specyfikacji ECMAScript. Język JavaScript składa się z elementów taich jak:
  • wartości
  • obiekty
  • zmienne
  • funkcje
  • literały
  • opcje porównywania

ZMIENNE W JS

- to takie połączenie z naszym wszechświatem wartości. Zmienna to nie wartość, ona tylko wskazuje na naszą wartość.
- LET - zmienne, których połączenie możemy przepinać, ale definiujemy tylko raz! np. let isHappy = false; isHappy = true; isHappy = 'tak';
- CONST - zmienne mające stałe połączenie z jedną wartością, np. const name = 'Maciek'; const age = 31;
- VAR - używane coraz rzadziej

Typy proste w JS (primitives)

- String - tekst, otaczamy ' lub " (zalecam ')
- Number - liczba
- Bigint - duża liczba ;) większa niż 253
- Boolean - true lub false
- Null - wartość celowo niezdefiniowana
- Undefined - brak definicji
- *symbol - tworzy "unikalne opakowanie"

- Aby dowiedzieć się jakiego typu jest zmienna używamy polecenia "typeof"
- Nie możemy zmieniać wartości primitive values w JS. To co zmieniamy to połączenie między nimi, a zmiennymi.

Funkcje

Funkcje wykorzystujemy wtedy, gdy mamy jakiś fragment kodu, który się nam powtarza.

OBIEKT

Obiekt jest strukturą danych. Własności obiektów opisujemy tworzą pary w postaci: klucz: wartość, np. levels: 357,
Klucz powinien byś unikalny w ramach obiektu, wartość - prymitywy/funkcje/zagnieżdzenia.

Wyrażenia logiczne

Porównania
  • > lewa strona większa od prawej
  • < lewa strona mniejsza od prawej
  • >= lewa strona większa / równa od prawej
  • <= lewa strona mniejsza / równa od prawej
  • == obie strony równe
  • === obie strony równe (sprawdza typeof)
  • != obie strony różne
  • !== obie strony różne (sprawdza typeof)
Falsy - 6 wartości: false 0 '', "", `` - pusty string null undefined NaN - not a number Jeżeli wyrażenie logiczne musi spełnić kilka warunków łączymy je symbolem && czyli AND (oba warunki muszą być prawdą) lub OR || (jeden z warunków jest prawdą). NOT - odwraca wartość wyrażenia logicznego, symbol wykrzyknik ! if (!('JavaScript' == 'Java')) { // ten kod się wykona }
