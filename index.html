Tydzień drugi

Nauka podstaw języka html, CSS oraz pojęcia związane z Java Script. Właśnie praktykuję używanie znaczników w języku html.

Semantyczny html - czyli mający znaczenie, nadajemy dodatkowe znaczenie treści, np. metaforyczne.

Aby stworzyć tzw. szklielet strony wciskamy ! a następnie tabulator. Kolejnym krokiem jest określenie języka strony, domyślnie jest to język angielski (en), w przypadku języka polskiego wpisujemy pl.

Aby sformatować kod wciskamy komendę comand+shift+p wpisujemy format.

Sprawdzamy poprawność kodu, ewentualne błędy przez walidator.

Tydzień trzeci

Robi się coraz trudniej...
Kolejne lekcje to:
- zapoznananie się z kolejnymi znacznikami kodu html.
- tworzenie hiperłączy na stronie oraz między stronami
- tworząc nazawy plików nie używamy spacji, ani polskich znaków, czy symboli. Najczęściej tworzymy nazwy kebab case'em, np. nauka-java-script.
- podstawy Java Scriptu
- zapoznanie się z działaniem terminala i podstawowe komendy poruszania się w konsoli

Pliki zawierające kod Java Script mają rozszerzenie . js oraz mają swój znacznik script, który umieszczamy na końcu strony, przed zamknięciem znacznika body
